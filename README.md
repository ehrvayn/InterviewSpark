# InterviewSpark ⚡

InterviewSpark is an AI-powered interview preparation platform designed to help users practice their interviewing skills through realistic simulations. Built with a modern tech stack, it provides real-time feedback, performance analytics, and skill breakdowns to bridge the gap between preparation and placement.

## 🚀 Features

- **AI Interview Simulations**: Practice Technical, Behavioral, and Case Study interviews.
- **Dynamic Analytics**: Track progress over time with a visual dashboard and skill breakdown (Clarity, Confidence, Relevance, etc.).
- **Smart Feedback**: Detailed scoring and insights on your responses.
- **Session History**: Review previous attempts, scores, and specific company-targeted practice.
- **Google Authentication**: Secure and easy access to your personalized dashboard.

## 🛠️ Tech Stack

### Frontend

### Backend
